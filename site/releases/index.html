Linux (Ubuntu/Debian) – Debian Repository

Add the signing key and package source for automatic updates:

curl -fsSL https://piyushdoorwar.github.io/transmux/debian/transmux.asc | gpg --dearmor | sudo tee /etc/apt/keyrings/transmux.gpg > /dev/null
echo "deb [signed-by=/etc/apt/keyrings/transmux.gpg] https://piyushdoorwar.github.io/transmux/debian stable main" | sudo tee /etc/apt/sources.list.d/transmux.list
sudo apt update
sudo apt install transmux

Linux (Ubuntu/Debian) – Direct .deb Install

Download the .deb file from the Versions tab, then install directly:

sudo apt install ~/Downloads/transmux_*.deb

Requires ffmpeg installed via sudo apt install ffmpeg

Windows – .exe Installer

Download the .exe installer from the Versions tab and run it. FFmpeg is bundled, no additional setup needed.
