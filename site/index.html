What it does

Convert once, play anywhere.

FFmpeg powered

Solid conversion through FFmpeg, bundled with release builds so it works out of the box.

Wide format support

MP4, WebM, MKV, AVI, MOV, MP3, AAC, FLAC, OGG, WAV, Opus — video and audio in one tool.

Subtitle handling

Embed subtitles in the output, extract tracks to SRT or ASS, or strip them entirely.

Live progress

Real-time speed and ETA feedback while converting, with cancel support at any point.

Latest release

Get Transmux for Ubuntu and Windows

Release builds are distributed as a Debian package for Linux and a single-file installer for Windows.
